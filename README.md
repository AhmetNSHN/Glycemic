# Glycemic

Projenin amacı kullanıcıları Gıdaların glisemik değerlerine ulaştırmaktır. Kullanıcılarda adminler gibi gıda ekleyerek kaktkıda bulunabilir, gıdalalrı seçerek glisemik indeks hesaplaması yapabilirler.
